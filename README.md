Este projeto implementa um sistema completo para construção de tabelas de distribuição de frequências utilizando Python puro.
O usuário fornece:

Uma sequência de valores numéricos em qualquer ordem

O número de classes desejado

O algoritmo realiza:

1-Ordenação dos dados;

2-Cálculo da amplitude total;

3-Cálculo do tamanho da classe;

4-Construção das faixas de classe;

5-Frequência absoluta;

6-Frequência relativa;

7-Frequência acumulada;

A saída final é exibida diretamente no terminal em formato de tabela alinhada e profissional.
